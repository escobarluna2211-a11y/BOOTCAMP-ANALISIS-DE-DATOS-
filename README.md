# 📊 Proyecto de Análisis de Datos (Bootcamp)

Este repositorio contiene un proyecto inicial de análisis de datos desarrollado como parte de un bootcamp. El objetivo es aplicar conceptos básicos de exploración, limpieza y análisis utilizando Python.

---

## 🚀 Descripción

Este proyecto se encuentra en una etapa temprana y tiene como propósito practicar habilidades fundamentales en análisis de datos, incluyendo:

* Manipulación de datos
* Exploración y limpieza
* Visualización básica
* Generación de insights iniciales

---

## 🛠️ Tecnologías utilizadas

* Python
* Google Colab
* Pandas
* Matplotlib / Seaborn (opcional, según avances)

---

## 📂 Estructura del proyecto

```
├── data/            # Datos utilizados (si aplica)
├── notebooks/       # Notebooks de análisis
├── src/             # Scripts (opcional)
└── README.md
```

---

## ▶️ Cómo usar este proyecto

1. Abre el proyecto en Google Colab
2. Ejecuta las celdas paso a paso
3. Revisa el análisis y las visualizaciones generadas

> Nota: Este proyecto está en desarrollo, por lo que algunas funcionalidades pueden estar incompletas.

---

## 📈 Estado del proyecto

🚧 En desarrollo (fase inicial)

Este proyecto irá evolucionando a medida que avance el bootcamp y se incorporen nuevos conceptos y técnicas.
